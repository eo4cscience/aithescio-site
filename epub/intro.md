# Aithesc.io Documentation

Welcome to the online documentation of **Aithesc.io**; a web application to investigate the morphology of the irish languague. The documentation is separated into two large sections: the theory and the implementation. The implementation concerns the use of the web application while the theory section concerns the linguistics of the irish language and the computational methods used to parse it. 

This platform integrates formal computational parsing with deep linguistic theory. In this documentation, you will find detailed explanations of Irish language typology, finite state methodologies, and the specific rules implemented in our parser.
