# Other Inflected Paradigms in Irish

## Overview
While Nouns and Verbs constitute the largest part of the Irish morphological system, several other parts of speech exhibit significant inflectional complexity. To build a complete eGrammar, these must also be modeled as Pynini feature spaces.

## 1. Adjectival Paradigms
Adjectives in Irish primarily agree with the noun they modify in gender, case, and number. Additionally, adjectives possess a comparative/superlative degree.

### Adjectival Feature Space
The grammatical space for adjectives overlaps significantly with nouns but adds the axis of "Degree".

| Feature | Code | Description | Valid Values |
| :--- | :--- | :--- | :--- |
| **Case** | `case` | Agreement Case. | `nom`, `gen`, `voc`, `dat` |
| **Number** | `number` | Agreement Number. | `sg`, `pl` |
| **Gender** | `gender` | Agreement Gender. | `masc`, `fem` |
| **Degree** | `degree` | Intensity/Comparison. | `pos` (Positive), `comp` (Comparative) |

**Note:** Irish does not have a separate "Superlative" form; the Comparative form is used in superlative contexts (e.g., *níos mó* → "bigger", *is mó* → "biggest").

---

## 2. Prepositional Paradigms (Prepositional Pronouns)
One of the most characteristic features of Irish is the **inflected preposition**. Rather than a separate preposition and pronoun (e.g., "with me"), Irish fuses them into a single word (*liom*).

### Prepositional Feature Space
Each inflected prepositional lexeme (like *ag*, *ar*, *le*) maps onto a 7-cell or 14-cell matrix depending on whether emphatic forms are included.

| Feature | Code | Description | Valid Values |
| :--- | :--- | :--- | :--- |
| **Person** | `person` | Grammatical person. | `P1`, `P2`, `P3` |
| **Number** | `number` | Grammatical number. | `sg`, `pl` |
| **Gender** | `gender` | Gender (P3sg only). | `Masc`, `Fem` |
| **Emphasis** | `emph` | Emphatic suffixation. | `None`, `Emph` |

**Example Matrix for *ag* (at/with):**
| Person/Num | Surface Form | Feature Vector |
| :--- | :--- | :--- |
| P1sg | *agam* | `[prep=ag] [person=P1] [number=sg]` |
| P2sg | *agat* | `[prep=ag] [person=P2] [number=sg]` |
| P3sg Masc | *aige* | `[prep=ag] [person=P3] [number=sg] [gender=Masc]` |
| P3sg Fem | *aici* | `[prep=ag] [person=P3] [number=sg] [gender=Fem]` |

---

## 3. Pronoun Paradigms
Personal and Possessive pronouns also exist within a feature space that mirrors the prepositional pronouns.

| Feature | Code | Description | Valid Values |
| :--- | :--- | :--- | :--- |
| **Type** | `type` | Pronoun class. | `Pers` (Personal), `Poss` (Possessive) |
| **Case** | `case` | Morphological Case. | `Subject`, `Object` (for Personal) |
| **Person** | `person` | Person. | `P1`, `P2`, `P3` |
| **Number** | `number` | Number. | `sg`, `pl` |

### Possessive Adjectives and Mutations
Unlike the personal pronouns, the **Possessive** feature space (`mo`, `do`, `a`, etc.) functions as a transduction trigger for initial mutations on the subsequent word.

- `[type=Poss] [person=P1] [number=sg]` (mo) → Triggers **Lenition**.
- `[type=Poss] [person=P1] [number=pl]` (ár) → Triggers **Eclipsis**.

## Summary of the Irish Grammatical Ecosystem
The union of these spaces—Verbal, Nominal, Adjectival, and Prepositional—forms the total "search space" for the FST engine. By mapping every axis with the `C`, `D`, and `P` prefixes established in our paradigm guides, we ensure a mathematically rigid but linguistically flexible model of the Irish language.
