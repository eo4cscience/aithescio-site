# Aithesc.io Documentation

Welcome to the online documentation of **Aithesc.io**; a web application to investigate the morphology of the irish languague. The documentation is separated into three large sections: the theory, the manual and the implementation code. The theory discusses the linguistics of the irish language and the computational methods used to parse it. The manual concerns the use of the web application while the implementation code section discusses the software development aspects of using Pynini and Finite State Transducers that powers the web application. 

This platform integrates formal computational parsing with deep linguistic theory and provides a publication method to share the results of the research. In this documentation, you will find detailed explanations of Irish language typology, finite state methodologies, and the specific rules implemented in our parser.
