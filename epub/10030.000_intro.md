# Aithesc.io - Code & Implementation

Welcome to the technical documentation for the Aithesc.io Morphological Parser.

This section covers the formal implementation details of the Irish language parser, including:

- **Finite State Transducers (FSTs)**: The underlying mathematical models used for morphology.
- **Pynini & Thrax**: The tools used to compile and execute the grammars.
- **API Integration**: How the backend serves linguistic analysis to the frontend.

## Implementation Roadmap

Currently, we are focusing on:
1.  **Noun Declension**: Formalizing the rules for Noun cases and mutations.
2.  **Verb Conjugation**: Implementing the complex verbal system of Irish.
3.  **Phonetic Interface**: Integration with the phonetics engine for surface-form generation.
